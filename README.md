# Retryn — Landing Page

The marketing and legal pages for **[Retryn](https://retryn.app)**, hosted via GitHub Pages.

## Pages

| File | URL | Description |
|---|---|---|
| `home` | `retryn.app` | Main marketing landing page |
| `privacy.html` | `retryn.app/privacy` | Privacy Policy |

## Deployment

This site is served automatically by **GitHub Pages** from the `main` branch.  
The custom domain is configured via the `CNAME` file → `retryn.app`.

Any commit pushed to `main` is live within ~60 seconds.

## Updating Content

- **Landing page copy** → edit `home`
- **Privacy Policy** → edit `privacy.html`
- **Custom domain** → edit `CNAME`
